# v0.1.0-beta.1 Release Notes

## What This Release Is

`v0.1.0-beta.1` is the source-first public beta for `big_boy_benchmarking`, the
official benchmarking and calibration repository for `state_collapser`.

The current component is:

```text
Big Boy Calibration / Smoke
```

The future component is:

```text
Benchmarking
```

This beta publishes the machinery and evidence needed to inspect environment
readiness, artifact contracts, tower construction, training health, calibrated
comparison readouts, and human-readable report generation. It is not a final
claim-bearing benchmark suite.

This release aalso includes the raw artifact bundle as release assets.

## Included Environment Families

- Counterpoint Symbolic v001.
- PlateSupport 5x5 Default v001.

## Included Evaluation And Readout Families

Counterpoint:

- first serious learning;
- one-third schema tower diagnostics;
- contraction fraction sweep diagnostics;
- noisy-rate contraction diagnostics;
- noisy-rate full-tower training diagnostic;
- second serious schema comparison;
- threshold frontier probe;
- small paired replicate probe.

PlateSupport:

- standard gauntlet.

## Key Bounded Smoke Result

The strongest current signal is the PlateSupport standard gauntlet correction
run. In that run, the selected iterated tower candidate beat the direct
baseline on the calibrated binary-success target and showed a coherent
action-filtering counter-signal: better mean total reward and zero invalid
concrete moves for the tower arm under the checked-in budget.

This supports continued benchmark design. It does not prove general tower
superiority, statistical significance at large budget, or broad robotics
performance.

## Source Install

```bash
git clone <repo-url>
cd big_boy_benchmarking
uv sync --group dev
uv run pytest
uv run python -m big_boy_benchmarking.cli --help
```

liftability semantics.

## Known Limitations

- This is calibration/smoke evidence, not final benchmarking.
- Raw artifacts are attached as GitHub release assets rather than tracked in
  git.
- The package is source-first; PyPI publication is deferred.
- Repository visibility is separate from the release record; if the repository
  is private, release access is limited to users with repository access.

## Compatibility

- `state_collapser v0.7.2` or newer compatible pointwise liftability semantics.
- Python `>=3.11,<3.13`.
- `uv` source workflow.
