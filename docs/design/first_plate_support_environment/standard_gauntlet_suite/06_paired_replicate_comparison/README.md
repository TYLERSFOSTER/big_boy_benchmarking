# Paired Replicate Comparison Design

## Purpose

This folder is for the first actual paired comparison stage in the PlateSupport
gauntlet.

## Starting Point

Counterpoint's paired replicate probe used matched seeds and a fixed threshold
surface to compare a flat/no-contraction baseline against a selected tower
condition. PlateSupport should preserve pairing discipline, but the baseline,
threshold, and success interpretation must be PlateSupport-specific.

## Design Question

What paired comparison gives the first honest evidence for or against
PlateSupport tower benefit without overclaiming?
