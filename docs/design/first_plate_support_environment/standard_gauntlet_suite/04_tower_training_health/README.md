# Tower Training Health Design

## Purpose

This folder is for a tower-only training-health diagnostic over selected
PlateSupport candidates.

## Starting Point

Counterpoint used this stage to verify that selected non-collapsed towers could
support concrete steps, lift/action realization, tier/controller traces, and
learner updates before making direct comparison claims.

## Design Question

What is the minimal PlateSupport tower-only training budget that proves runtime
and learner health without claiming superiority?
