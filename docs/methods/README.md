# Methods

Human-facing method notes live here.

These pages explain artifact contracts, mode semantics, metric channels, seed
bundles, timing boundaries, readout discipline, and statistical summaries. The
machine-readable schemas, manifests, and event tables remain the source of
truth for executed runs.

Method docs support all three workflow stages:

- environment construction needs legality, reward, mask, timing, and diagnostic
  method notes;
- evaluation construction needs artifact contracts, seed policy, modes,
  expected-file policy, and statistics;
- readout processing needs timing/readout discipline and field/status
  translation.

The protocols that bind these methods together live under
`docs/prime_directive/`.
