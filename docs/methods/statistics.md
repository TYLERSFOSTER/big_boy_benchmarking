# Statistics

The first shared machinery slice only provides small replicate-level summary
helpers.

It does not establish serious benchmark claims, final statistical reporting, or
large experiment conclusions. Future serious result claims need their own
experiment design and review.

Bootstrap helpers must operate over replicate-level values, use explicit
random seeds, and reject empty data clearly.
