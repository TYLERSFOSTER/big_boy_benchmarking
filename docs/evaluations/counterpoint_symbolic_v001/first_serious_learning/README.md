# Counterpoint First Serious Learning Evaluation

This checked-in page is the stable guide for the first serious learning
evaluation docs. It is not the status page for a particular local run.

Generated readouts are written under the artifact root by default:

```text
<artifact-root>/evaluations/counterpoint_first_serious_learning_v001/docs/
```

Primary serious fixture:

```text
counterpoint_symbolic_n3_small_v001
```

Tiny fixture:

```text
counterpoint_symbolic_n3_tiny_v001
```

The tiny fixture is for smoke and CLI validation only.

Claim boundary: this evaluation can compare direct and tower-control learning
arms under the recorded `tensor_available_disabled` condition. It does not make
a tensor-enabled, GPU, musical-quality, or general method claim.
