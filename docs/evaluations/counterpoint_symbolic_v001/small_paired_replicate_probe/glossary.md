# Glossary

- Schema 0: the matched no-contraction condition.
- Schema 1: selected noisy-rate quotient candidate using full-iterated tower construction.
- Seed bundle: the shared random-seed identity used by both schema arms inside a pair.
- Post-hit margin: reward above or below the locked threshold inside the first sustained-hit window.
- Claim-blocked pair: a pair that cannot support comparison because one arm failed, missed sustained hit, or has incomplete artifacts.
