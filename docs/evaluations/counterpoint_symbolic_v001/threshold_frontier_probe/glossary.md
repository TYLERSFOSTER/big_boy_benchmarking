# Glossary

- threshold frontier: the highest tested reward threshold where an arm still reaches sustained-hit.
- sustained-hit: the configured persistence rule over episode total reward, here 4 of 5 windows by default.
- Schema 0: no-contraction total-space condition.
- Schema 1: corrected full-iterated noisy-rate tower condition.
- next-measure probe: evidence strong enough to steer the next run, not a final statistical comparison.
