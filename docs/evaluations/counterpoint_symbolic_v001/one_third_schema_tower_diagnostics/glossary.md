# Glossary

- ABC: upstream active-tier controller behavior that decides whether to lift, descend, train, explore, or exploit/execute.
- source-local one-third: contraction assignment computed separately for each source state's outgoing edges.
- selected tier: the lowest executable unclosed tier selected by upstream ABC helper logic.
- executable tier: a tier whose current quotient state has outgoing action cells available.
- concrete step: an abstract tower action successfully lifted to a base counterpoint transition.
- near full collapse: tier-1 largest state fiber share is at least 0.90.
